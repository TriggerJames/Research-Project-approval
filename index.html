Convert your documents, audio, and video files with ease!

Features

Document Conversion

Convert documents to PDF, DOCX, and TXT formats.

Audio Conversion

Convert audio files to MP3, WAV, and other formats.

Video Conversion

Convert video files to MP4, AVI, and other formats.

About

This project was inspired by the need for an easy-to-use file conversion tool. Developed as a portfolio project for ALX Africa, it aims to simplify the process of converting various file formats. For more information, visit my GitHub Repository.
